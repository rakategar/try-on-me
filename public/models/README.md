# 3D Models Directory

Place your T-shirt 3D models here.

## Recommended Format

- `.glb` (preferred) or `.gltf`
- Poly count: < 10,000 triangles
- UV mapped

## Example Files

- tshirt-basic.glb - Basic t-shirt model
- tshirt-vneck.glb - V-neck variant
- tshirt-longsleeve.glb - Long sleeve variant

## Where to Get Models

1. **Blender** - Create your own
2. **Sketchfab** - Download free models
3. **TurboSquid** - Premium models
4. **CGTrader** - Free & paid models
